# ADL-TEST-GUARD

Application-layer DoS/DDoS testing framework for controlled L7 experimentation.

## Features
- TA1: Volumetric HTTP Flood (Locust)
- TA2: Slow attacks (Slowloris, SlowHTTPTest)
- TA3: Application Logic Abuse
- TA4: Service/API Abuse
- TA5: Custom Python L7 Floods

## Design
Single active test execution to ensure reproducibility and clean telemetry.
